자바스크립트에서 function을 사용할 필요가 없는 이유.
function은 다양한 목적에 두루 사용할 수 있는 특징이 있다 (ex 메서드로 사용, 함수로 사용, 생성자함수로 사용 -> 장점
하지만 function은 동작면에서 프로그램을 무겁게하고 상황에 따라서 개발자가 추가로 번거로운 작업을
해줘야 하거나 안전 장치를 마련할 필요가 있다. -> 단점 

위 단점들로 인해 협업 개발에 있어서 어려움이 생길 수 있으며 아래의 대체 방식을 이용하는 것을 권장 한다.

함수로 사용하고자 할 때 : arrow함수 () => {}, arguments, caller부분에서 차이가 있으며 
상대적으로 가벼움 invoke 시켜야 값을 알 수 있음(getter 호출 필요)

생성자함수로 쓰고자 할 때 : class 사용 / arguments에서 차이가 존재
메서드로 쓰고자 할 때 : method 축약형 / arrow함수처럼 가볍지만 바인딩 가능, obj1.method()와 같이 사용 불가능
